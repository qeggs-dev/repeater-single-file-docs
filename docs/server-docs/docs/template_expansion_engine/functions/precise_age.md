# Precise Age Function

获取指定生日的精确年龄

- `precise_age`
  - Params:
    - `birthday_year` (int): 出生年份
    - `birthday_month` (int): 出生月份
    - `birthday_day` (int): 出生日
    - `birthday_hour` (int): 出生小时
    - `birthday_minute` (int): 出生分钟
    - `birthday_second` (int): 出生秒数
  - Returns:
    - (float):
     - float: 精确年龄
